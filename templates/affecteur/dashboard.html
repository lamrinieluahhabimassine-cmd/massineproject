Guide d'Utilisation
Comment affecter des dossiers en masse:
  1. Téléchargez le modèle Excel en cliquant sur le bouton vert ci-dessus
  2. Remplissez le fichier avec les informations des dossiers:
    • file_number: Numéro unique du dossier (ex: VOC-2025-001)
    • receipt_date: Date de réception (format: YYYY-MM-DD)
    • importer: Nom de l'importateur
    • exporter: Nom de l'exportateur
    • country: Pays d'origine
    • route: Route (A, B ou C)
    • sor_number: Numéro SOR (requis pour Route B)
    • sol_number: Numéro SOL (requis pour Route C)
    • user_email: Email de l'utilisateur à qui affecter le dossier
  3. Téléversez le fichier en cliquant sur "Téléverser Maintenant"
  4. Les dossiers seront créés avec le statut en attente d'évaluation
  5. Les utilisateurs recevront une notification automatique
Important:
  • Les emails des utilisateurs doivent exister dans le système
  • Les numéros de dossiers doivent être uniques
  • Route B nécessite un numéro SOR
  • Route C nécessite un numéro SOL
